# Borrowed Items App

This folder contains a small demo application with a backend and a React frontend.

Use `start_app.sh` (Unix/macOS) or `start_app.bat` (Windows) from the project
root to install dependencies and launch both services at once.

Alternatively you can run the server and client individually:

```bash
cd borrowed_items_app/server && npm install && npm start
cd borrowed_items_app/client && npm install && npm start
```
