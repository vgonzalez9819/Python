# Borrowed Items Client

A simple React front-end for the item check-in app. Requires Node.js and npm.

## Usage

Install dependencies and start the development server:

```bash
npm install
npm start
```

Alternatively run `start_app.sh` (Unix/macOS) or `start_app.bat` (Windows)
from the project root to launch the entire application in one step.

The app proxies API requests to `localhost:3001` where the server is expected to run.
